# 💰 AI-Powered Personal Finance Advisor

An intelligent finance management system that combines Machine Learning, Deep Learning, NLP, and Agentic AI to predict future spending, analyze transactions, and provide personalized financial recommendations.

## 🚀 Features

* 📊 **Spending Forecasting** using LSTM-based deep learning models.
* 🧠 **Expense Classification** using Random Forest with **96.8% test accuracy**.
* 📩 **SMS Transaction Parser** to extract transaction details from bank SMS messages.
* 🤖 **Financial Agent** for budget monitoring, risk assessment, and recommendations.
* 📈 **Interactive Dashboard** built with Streamlit and React for financial insights and visualization.

## 🛠️ Tech Stack

* Python
* Scikit-learn
* TensorFlow / Keras
* Pandas, NumPy
* Streamlit
* React.js
